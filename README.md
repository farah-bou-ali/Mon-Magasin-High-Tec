# 💻 Mon Magasin High-Tech

## 🚀 Comment exécuter le projet ?
1. **Serveur Web** : Utilisez MAMP ou XAMPP.
2. **Base de données** : 
   - Créez une base de données nommée `panier_db` dans PostgreSQL.
   - Importez le fichier `database.sql` fourni dans ce dépôt.
3. **Configuration** : Vérifiez le fichier `db.php` pour les identifiants de connexion.
4. **Accès** : Placez les fichiers dans `htdocs` et allez sur `localhost/votre_projet/index.php`.
